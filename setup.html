Setup¶

Skoot depends on several prominent Python packages:

  • Numpy
  • SciPy
  • Scikit-learn (>= 0.18)
  • Pandas

To install, simply use pip install skoot.


Building from source¶

If you hope to contribute to the package, you’ll need to build from source. Since there are C & Fortran dependencies, the compilation procedure adds some new dependencies:

  • Cython
  • A Fortran compiler (see following sections)

Linux/Unix machines (Mac OS)¶

Building on a unix machine is easier than building on Windows. All you need is gcc, g++ and gfortran. These can be downloaded via several methods:

Mac OS:

$ brew install gcc

Linux:

$ apt-get install gcc
$ apt-get install g++
$ apt-get install gfortran

After you have a Fortran compiler, you can build the package. To build in development mode (to be able to run tests):

$ python setup.py develop

To install the egg in your site-packages:

$ python setup.py install

Windows¶

tl;dr: don’t do it. It’s painful… but it can be done. Like with Linux/Unix machines (Mac OS), you’ll need a Fortran compiler. You can either use gfortran, mingw or MSVC. After you have a Fortran compiler, you can build the package. To build in development mode (to be able to run tests), you can use the same command as with Linux:

$ python setup.py develop

You can also install from a wheel, e.g.:

$ python setup.py bdist_wheel bdist_wininst
$ pip wheel --wheel-dir=dist .
$ pip install dist/skoot-...-win32.whl
